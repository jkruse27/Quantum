How to Play

The main goal of the game is to return the qubit to its base state (|0\>). To do so, you can apply any of the gates which will be explained below. You can reset the qubit to its beggining state, before you applied any qubits. You can also begin a new game by clicking on the "new state" or "random state" buttons. The difference between the 2 is that when you create a "new state", the state will be created from the base state by applying the same gates you have access to , while the "random state" generates a completely random state. You can choose the number of gates applied to the new state in the text box.

Gates

X

This gate flips the state of the qubit 180 degrees on the X axis. It can be represented by the ... matrix, taking the a|0> + b|1> state to the b|0> + a|1> state.

Y

This gate flips the state of the qubit 180 degrees on the X axis. It can be represented by the ... matrix, taking the a|0> + b|1> state to the b|0> + a|1> state.

Z

This gate rotates the qubit 180 degrees on the Z axis. It can be represented by the ... matrix, taking the a|0> + b|1> state to the a|0> - b|1> state.

H

This gate flips the state of the qubit 180 degrees on the X axis. It can be represented by the ... matrix, taking the a|0> + b|1> state to the b|0> + a|1> state.

S

This gate flips the state of the qubit 180 degrees on the X axis. It can be represented by the ... matrix, taking the a|0> + b|1> state to the b|0> + a|1> state.

T

This gate flips the state of the qubit 180 degrees on the X axis. It can be represented by the ... matrix, taking the a|0> + b|1> state to the b|0> + a|1> state.
